# CEO Briefing Generator

Automated weekly business intelligence reports.

## Setup

### 1. Install Dependencies

```bash
pip install python-dateutil
```

### 2. Schedule Weekly Run

```powershell
# Windows - Every Monday at 8 AM
schtasks /Create /TN "AI_Employee_CEO_Briefing" /TR "python ceo_briefing.py C:\path\to\vault" /SC WEEKLY /D MON /ST 08:00
```

```bash
# Linux/Mac - Every Monday at 8 AM
crontab -e
# Add: 0 8 * * 1 python /path/to/ceo_briefing.py /path/to/vault
```

## Usage

```bash
# Generate briefing for last week
python ceo_briefing.py /path/to/vault

# Generate for specific date range
python ceo_briefing.py /path/to/vault --start 2026-02-01 --end 2026-02-28

# Preview without saving
python ceo_briefing.py /path/to/vault --preview
```

## Output

Generates `Briefings/YYYY-MM-DD_CEO_Briefing.md`:

```markdown

**Period:** Feb 24 - Mar 1, 2026

## Executive Summary

Strong week with revenue ahead of target. One bottleneck identified.

## Revenue

- **This Week:** $2,450
- **MTD:** $4,500 (45% of $10,000 target)
- **Trend:** On track

## Completed Tasks

- [x] Client A invoice sent and paid
- [x] Project Alpha milestone delivered
- [x] Weekly social media scheduled

## Bottlenecks

| Task | Expected | Actual | Delay |
|------|----------|--------|-------|
| Client B proposal | 2 days | 5 days | +3 days |

## Proactive Suggestions

### Cost Optimization

  - [ACTION] Cancel subscription?

### Upcoming Deadlines

*Generated by AI Employee v0.1*
```

## Metrics Tracked

| Metric | Source |
|--------|--------|
| Revenue | Accounting/ transactions |
| Tasks Completed | Done/ folder count |
| Response Time | Email/WhatsApp timestamps |
| Bottlenecks | Plan.md completion times |
| Subscription Costs | Recurring transactions |
