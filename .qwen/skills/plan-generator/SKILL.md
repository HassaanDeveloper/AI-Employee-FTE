# Plan Generator

Create structured action plans for AI Employee tasks.

## Usage

```bash
# Generate a plan for a task
python scripts/plan_generator.py /path/to/vault \
  --task "Process invoice request from Client A" \
  --output "Plans/PLAN_invoice_client_a.md"

# Generate from existing action file
python scripts/plan_generator.py /path/to/vault \
  --from-file "Needs_Action/EMAIL_12345.md"
```

## Plan Template

Generated plans follow this structure:

```markdown
---
type: plan
created: 2026-03-01T10:30:00Z
status: in_progress
task_id: TASK_001
priority: normal
estimated_steps: 5
---

# Plan: [Task Name]

## Objective
Clear description of what needs to be accomplished.

## Steps

- [ ] Step 1: First action
- [ ] Step 2: Second action
- [ ] Step 3: Third action

## Resources
- [[Company_Handbook]] - Rules to follow
- [[Business_Goals]] - Relevant goals

## Notes
Additional context and considerations.

---
*Generated by Plan Generator*
```

## Integration

Plans are automatically created by the Orchestrator when processing
complex tasks. The AI references the plan during execution and checks
off completed steps.
